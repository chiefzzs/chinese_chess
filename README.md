# deepseek-chess

希望达成可deepseek下象棋。


通过纯文本引导deepseek写的一个简单的中国象棋引擎。

**棋子含义**：

将/帅: K/k, 仕/士: S/s, 相/象: X/x, 傌/马: M/m, 俥/车: C/c, 炮/砲: P/p, 兵/卒: B/z
红方（大写）先走，黑方（小写）为 AI。

输入移动指令，例如：7 7 7 4 表示将红炮(P)从(7行,7列)移动到(7行,4列)。

![输出信息](./qipan.jpg)
